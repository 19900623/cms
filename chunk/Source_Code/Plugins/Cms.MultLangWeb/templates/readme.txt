” ${siteinfo(yes,300)} “
	网站信息
	参数1：是否显示图片,值：[ yes|no ]

“ ${alist(tag,8,true,flags)} ”
   文档列表
   参数:
   1.分类标签
   2.显示条数
   3.是否显示图片
   3.参数

${ac(id,8)}
  输出文档内容
  参数：
  1: 文档ID或别名
  2: 内容长度，可为空，如：${ac(id,)}


${palist(news,1,10,top,p)}
 输出分页文档列表
 参数：
 1.分类Tag
 2.页码,从1开始
 3.每页显示条数
 4.参数
 5.单项的Html元素，默认为"P"
 
${slist(${key},${pageIndex},10,topcv)}
搜索结果列表
参数：
1.搜索关键词
2.页码
3.每页显示条数
4.参数


${clist(param,target)}
 输出分类列表(UL)
 参数：
 1.可为空，返回所有分类;
   分类Tag则返回其子分类;
   数字则返回指定模块的分类。
 2.链接打开窗口
